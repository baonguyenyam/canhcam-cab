## This is a Tool for WebApp Front-End with All-in-One Gulp task 

##### Template 

* Jade Template
* Nunjucks Template

##### Scripts 

* Babel
* Typescript
* Coffee 

##### Styles

* SASS/SCSS
* LESS
* Stylus
* CSS

##### Image Tools

* SVG
* Real Favicon

##### Build Tools

* CSSMin
* Uglify
* HTML minify
* ImageMin
* Concat 

##### Test Tools

* Browserify
* BrowserSync
* Phantom
* Karma


### Install Project 

Windows: run file `install.bat`

MAC/Ubuntu: run file `sh install.sh` 

### Update Project

Windows: run file `update.bat`

MAC/Ubuntu: run file `sh update.sh` 

### Upgrade & Check Missing package

run `npm run update`

### Config 

Config file `config.yml`

### Working in Project

Run: run `npm run app`

Tools: run `npm run tool`
